# Bus-Management-System
A web-based application that streamlines public transport operations with features like user registration, QR code generation, fare management, and journey tracking. Passengers can scan QR codes to start and stop trips, while the system calculates travel distance and automatically deducts fares from their card balance.

A comprehensive bus management system designed to streamline public transport operations.  
It provides an **admin panel** for managing users, generating QR codes, fare management, and tracking passengers,  
while the **user interface** enables passengers to scan QR codes to start and stop journeys.  
The system calculates travel distance and automatically deducts the fare from the user’s card balance.  

---

##  Features
-  User & Admin registration/login  
-  QR code generation for travel cards  
-  Recharge and fare deduction system  
-  Admin dashboard to manage users and journeys  
-  Automatic travel distance & fare calculation  
-  Passenger QR scan to start/stop journeys  

---

## Technologies Used
- **Frontend**: HTML, CSS, JavaScript  
- **Backend**: PHP  
- **Database**: MySQL  
- **Libraries**: FPDF, PHP QR Code Generator
